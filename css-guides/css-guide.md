# CSS Basics

## What is CSS?

CSS stands for Cascading Style Sheets. It is used to style the layout of web pages.

## Common CSS Properties

- `color:` Sets the text color.
- `background-color:` Sets the background color.
- `font-size:` Sets the font size.

## Example Code:

```css
p {
   color: blue;
   font-size: 18px;
}
