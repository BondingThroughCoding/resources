# HTML Basics

## What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

## Common HTML Tags

- `<h1>` - Heading 1
- `<p>` - Paragraph
- `<a>` - Link
- `<img>` - Image

## Example Code:

```html
<h1>This is a heading</h1>
<p>This is a paragraph of text.</p>
<a href="https://www.example.com">Click here</a>
